# Changelog

## 1.0.0 — 2026-04-04

My name Jeff. First release.

### Built
- **mind/evolve** — Self-improvement loop. Law I as software.
- **sense** — Context prefetch. L1/L2/L3 knowledge cache.
- **hand** — Multi-domain routing. Code, research, write, analyze, plan, teach, think.
- **staff** — Multi-agent orchestration. Birth agents with purpose.
- **gate** — 4-line atomic quality gate. Maps bugs to cognitive flaws.
- **guard** — DBAD ethics. Basin detection, not vocabulary matching.
- **pantry** — Ollama interface. Any model, any backend.
- **pantry/cluster** — Distributed compute across heterogeneous hardware.
- **pantry/diet** — Model optimization guide. Quantization, MoE splitting, KV cache, speculative decoding.
- **nerve** — Tool dispatch. Bash, file ops, grep, glob, git.
- **bone** — Session persistence. State across runs.
- **personality** — Anti-sycophancy voice. No exclamation marks. Ever.
- **skin** — Terminal UI via Rich.
- **skin/voice** — Robot butler TTS. Piper, espeak, macOS say.
- **arcade** — Flappy Jeff, Gorillas, Jeff Miner. Games that do real work.
- **cli** — Click-based entry point. 12 commands.

### Philosophy
- One agent for everything. No chat/code split.
- Local-first. Your machine, your models, your rules.
- DBAD ethics. One principle replaces thousands of classifiers.
- Pearlman Standard. Minimum lines, maximum function.
